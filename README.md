# Daengs-Club-Guest-Book

Tkinter 기반의 데스크톱 도구로 미용 전·후 사진과 고객/강아지 정보를 간단히 정리합니다. 드래그 앤드 드롭으로 사진을 넣고, 입력한 정보는 엑셀 파일(`customer_data.xlsx`)과 고객별 폴더(`고객사진/<고객번호 - 보호자 - 강아지>`)에 저장됩니다.

## 주요 기능
- 미용 전/후 사진을 드래그 앤드 드롭하거나 파일 선택으로 불러오기 (JPG/PNG)
- 고객번호·보호자/강아지 이름·품종·스타일·결제 상태·요청/특이사항/애프터 케어 입력
- 파일명을 자동으로 정리해 고객별 폴더에 저장하고, 기록을 엑셀에 추가
- 품종 리스트를 `breeds.txt`에서 읽어와 콤보박스로 선택, "기타(직접입력)" 지원
- 입력 오류 시 메시지 박스 안내 및 오류 로그 복사 기능 제공

## 실행 방법
1. 가상환경을 만들고 의존성을 설치합니다.
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
2. `breeds.txt`에 품종 목록을 한 줄씩 적어둡니다. 기본 파일이 제공되며, 필요한 경우 수정하면 됩니다.
3. 애플리케이션을 실행합니다.
   ```bash
   python main.py
   ```
4. 사진과 정보를 입력 후 **실행**을 누르면 고객별 폴더와 엑셀에 기록이 추가됩니다.

## 출력 위치
- 사진: `고객사진/<고객번호 - 보호자 이름 - 강아지 이름>/<고객번호 - 보호자 - 강아지 - 시각> - 미용전/후.ext`
- 기록: `customer_data.xlsx` (프로젝트 루트에 자동 생성)

## 주의사항
- 파일명 길이가 100자를 넘으면 저장이 거절됩니다. 이름이나 고객번호를 줄여주세요.
- 결제금액에는 숫자만 입력할 수 있으며, 입력란을 벗어나면 자동으로 천 단위 콤마가 들어갑니다.
- 사진은 JPG/JPEG/PNG만 지원합니다.
